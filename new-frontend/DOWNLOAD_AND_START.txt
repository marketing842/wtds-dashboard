Issue: Port 3000 already in use
Fix: pnpm dev will automatically use port 3001, or:
     kill $(lsof -t -i :3000)
     pnpm dev

Issue: Dependencies not installing
Fix: rm -rf node_modules pnpm-lock.yaml
     pnpm install

Issue: Hot reload not working
Fix: Stop server (Ctrl+C)
     pnpm dev
     Refresh browser

Issue: TypeScript errors
Fix: Check error messages in terminal
     See: tsconfig.json for configuration
     Run: pnpm type-check
